
# Tic - Tac - Toe application

Created a tic-tac-toe game using ReactJs

## Demo

https://tic-tac-toe-231.netlify.app/


## Run Locally

Clone the project

```bash
  git clone https://github.com/CodeinIP/Tic-Tac-Toe.git
```

Go to the project directory

```bash
  cd Tic-Tac-Toe
```

Install dependencies 

```bash
  npm install
```
or
```bash
pnpm install
```

Start the server

```bash
  npm run start
```
